## Project Overview

Homify is a modern real estate rental platform built with a decoupled architecture. It features a robust Django backend providing RESTful APIs and a dynamic React frontend providing a seamless user experience.

Key features include:
- Property listings with detailed information and photos
- Interactive maps for property locations
- User authentication and role management (Tenants, Landlords, Admins)
- Real-time messaging between users
- Favorites and saved properties
- Property reporting and moderation
